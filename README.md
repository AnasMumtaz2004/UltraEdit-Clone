# UltraEdit Landing Page Clone

## Description

This project is a **frontend clone of the UltraEdit download page**, built using **HTML and CSS**. It replicates the layout of a professional software landing page, including navigation, hero section, and a responsive download grid.

The goal of this project is to practice **layout design, responsive UI, and clean styling techniques** without using any frameworks.

---

## Features

* Clean and modern UI design
* Fully responsive layout using CSS Grid and Flexbox
* Navigation bar with dropdown indicators
* Hero section with call-to-action button
* Download section with multiple language options
* Styled buttons for different download versions (32-bit / 64-bit)

---

## Tech Stack

* **HTML5** – Structure of the webpage 
* **CSS3** – Styling and layout
